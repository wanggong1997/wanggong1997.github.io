但是我每次都要输sh ~/desktop/father/demon.sh实在太复杂了 有没有一种方法我直接输入demon就可以执行该脚本

那么首先将 ~/father 添加到 PATH 里

PATH 的作用
你每次在 Bash 里面输入一个命令时（比如 ls、cp、demo），Bash 都会去 PATH 列表里面寻找对应的文件，如果找到了就执行。

i. cd ~/father; pwd 得到 father 的绝对路径

好那么/c/Users/wanggong/desktop/father就是我的绝对路径了
ii .接下来创建 ~/.bashrc：touch ~/.bashrc(“.bashrc”呢就像是一个工作环境你要用他来创建你的快捷方式 这里呢我也是希望用来来创建一个快捷方式最终达到我的目的)
iii .编辑 ~/.bashrc：start ~/.bashrc，在最后一行添加 export PATH="local的绝对路径:$PATH"
这边我就可以换成 export PATH="/c/Users/wanggong/desktop/father:$PATH"
iv.source ~/.bashrc
v.之前你要运行 sh ~/desktop/father/demon.sh，现在你只需要运行 demo.txt 就行了（想想为什么，道理显而易见）
6.demo.txt 的后缀 .txt 很无聊，删掉它mv ~/local/demo.txt ~/local/demo
现在你只要运行 demo 就能执行该脚本了。
