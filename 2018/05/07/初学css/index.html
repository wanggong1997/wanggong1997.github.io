初学css

Posted on 2018-05-07

CSS是(Cascading Style Sheet)的缩写，也被称为层叠样式单；主要用作网页风格设计，包括字体大小，颜色，以及元素的精确定位等。

1.如何让CSS和html5有关联呢/

引入外部css样式单文件，在html5的<head.../>元素内插入<link.../>子元素:

clearfix::after{
content: ‘’;
display:block;
clear:both;
}
`

注意添加float:left的子元素 其直属母元素上一定要加clearfix

5.当你不知道自己<div>里面框的是什么的时候你可以在这个div里面加一个style="border:1px solid red"这个属性会让你直接看到div的框架。

6 我学习css中所用到的属性及其含义:

padding:20px 0px 20px 0px(padding是内边距 后面的数字意思分别是 上 右 下 左)

margin外边距 用法跟padding一样也是后面加所需要的数字

text-decoration:none(这个是样式 默认的样式是有下划线underline的 这个呢就是帮助你去掉下划线 也还有overline上划线等等)

text-align:center(文字居中的意思)

font-weight:bold(字体加粗)

font-family:字体样式(意思的选择你想要的字体样式)

font-size(字体大小tiaojie)

display-block(脱离文档流)

background-position:center center(背景图片居中)

background-size:cover(背景图片按比例缩放)

mix-weight:100px(最大宽度不超过100px)

margin-left:auto ; margin-right:auto;(可以实现水平居中)
